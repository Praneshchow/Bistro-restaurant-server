# Bistro-restaurant-server


## Server setup:

Initialize: `npm init -y`

Install express, cors, dotnet, mongodb: `npm i express cors dotenv mongodb`

### Run the Server: `nodemon index.js`
 


Study info: 

-----------------------
Naming Convention
-----------------------
* users: useCollection
* app.get('/users')
* app.get('/users/:id')
* app.post('/users')
* app.patch('/users/:id')
* app.put('/users/:id')
* app.delete('/users/:id')
